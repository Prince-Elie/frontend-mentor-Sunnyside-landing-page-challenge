# Sunnyside Landing Page

Un projet d'entraînement basé sur un défi **Frontend Mentor**. Une page de présentation statique pour mettre en pratique les fondamentaux du web.

## Aperçu

![Screenshot du projet](./screenshot.png)

## À propos du projet

Ce projet fait partie de mon parcours d'apprentissage via les défis Frontend Mentor. L'objectif est de mettre en pratique les compétences en HTML, CSS et JavaScript en construisant des pages web à partir de designs fournis.

## Caractéristiques

- **Design responsive** : Adaptation fluide sur mobiles, tablettes et écrans de bureau
- **Navigation mobile** : Menu hamburger pour petits écrans
- **Mise en page moderne** : Utilisation de Flexbox et Grid
- **Images responsive** : Format optimisé pour différentes résolutions

## Structure du projet

```
├── index.html       # Page HTML principale
├── style.css        # Feuilles de style
├── script.js        # Logique JavaScript
├── images/          # Images pour desktop et mobile
└── README.md        # Ce fichier
```

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript vanilla
- Font Awesome (icônes)
- Google Fonts (Barlow, Fraunces)

## Installation et utilisation

1. Clonez le projet ou téléchargez les fichiers
2. Ouvrez `index.html` dans votre navigateur
3. Profitez de la visite! 🎨
